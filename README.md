# Othello

CLIで遊べるオセロ（リバーシ）ゲームです。Go言語で実装しています。

## フォルダ構成

```
Othello/
├── src/        # オセロアプリのソースコード
├── tests/      # srcに対するテストコード
├── doc/        # 仕様書およびドキュメント
└── README.md
```

### 各ディレクトリの説明

- **src/**: オセロアプリの実装コードを管理します
- **tests/**: src配下のコードに対するユニットテスト・統合テストを管理します
- **doc/**: 仕様書、設計書、その他ドキュメントを管理します
